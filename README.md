# @amlplugins/vercel-ai-azure

Native `@ai-sdk/azure` SDK re-exported under the `@amlplugins` namespace.

> Vercel AI SDK Azure OpenAI provider — GPT-4o/o1/o3 via Azure deployments with Entra ID auth.

## Install

```bash
npm install @amlplugins/vercel-ai-azure
```

## Use

```ts
import { /* named exports from @ai-sdk/azure */ } from "@amlplugins/vercel-ai-azure";
// or
import sdk from "@amlplugins/vercel-ai-azure";
```

This package additionally re-exports `ai` under the `secondary` namespace.

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
